# NEXUS
Academic project of BDA(Big Data Analytics)


Problem Statement:
Generate synthetic Bitcoin data, validate anomalies such as missing or duplicate records, and train regression models to predict closing prices. The objective is to demonstrate predictive modeling with financial time-series data.
